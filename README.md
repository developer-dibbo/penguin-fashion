Penguin Fashion – E-commerce Landing Page

Penguin Fashion is a stylish and responsive e-commerce landing page designed for showcasing fashionable Men’s and Women’s Jackets.
This project was built using HTML5 and CSS3 with a focus on clean layout, modern UI, and responsive design.

🔥 Features
Fully responsive design
Modern and clean user interface
Product showcase sections
Stylish hero banner
Easy-to-navigate layout
Beginner-friendly code structure
